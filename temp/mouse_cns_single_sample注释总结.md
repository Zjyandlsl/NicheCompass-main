## 🔑 核心概念解释

### 1. Gene Program (GP) - 基因程序

```
GP = 一组功能相关的基因

来源基因 (sources) ──发送信号──> 目标基因 (targets)
  [配体/酶]                            [受体/传感器/靶基因]

示例: VEGFA_combined_GP
  来源: VEGFA (血管内皮生长因子)
  目标: FLT1, KDR, FLT4 (VEGF 受体)
  功能: 血管生成
```

### 2. GP Mask - GP 掩码

```
GP 掩码 = GP 的二进制矩阵表示

       GP1  GP2  GP3  ...
  GeneA  1    0    1
  GeneB  0    1    0
  GeneC  1    1    0
  ...

值 = 1: 该基因属于该 GP
值 = 0: 该基因不属于该 GP
```

### 3. Niche - 细胞生态位

```
生态位 = 空间上一致且功能相似的细胞群体

特征:
  1. 空间上相邻
  2. 基因表达相似
  3. GP 活性相似
  4. 执行相似功能

示例:
  - 血管周围生态位
  - 免疫浸润生态位
  - 代谢活跃生态位
```

### 4. Latent Space - 潜在空间

```
潜在空间 = 模型学习的低维表示

高维基因表达 (1022 维)
       ↓
   [编码器]
       ↓
潜在空间 (~300 维)
  ├─ 维度 1: GP1 活性 (如血管生成)
  ├─ 维度 2: GP2 活性 (如免疫反应)
  └─ ...
       ↓
   [解码器]
       ↓
重构基因表达 + 重构空间邻居图
```

### 第 2 部分：Prepare Model Training (准备训练数据)

#### 2.1 创建 GP 掩码

**步骤 1: 从 OmniPath 提取配体-受体 GP**
```
OmniPath 数据库
  ↓
约 2500 个 GP (配体-受体)
```

**步骤 2: 从 NicheNet 提取相互作用 GP**
```
NicheNet 数据库
  ↓
约 1500 个 GP (配体-受体-靶基因)
```

**步骤 3: 从 MEBOCOST 提取酶-传感器 GP**
```
MEBOCOST 数据库
  ↓
约 150 个 GP (酶-传感器)
```

**步骤 4: 过滤和合并**
```
原始: ~4150 个 GP
  ↓ 过滤 (移除低质量 GP)
  ↓ 合并 (合并重叠度 > 90% 的 GP)
最终: ~297 个高质量 GP
```

#### 2.2 加载数据并计算空间邻居图

**读取数据**
```
.h5ad 文件 (AnnData 格式)
  ├─ adata.X: 基因表达矩阵
  ├─ adata.obs: 细胞注释
  ├─ adata.var: 基因注释
  └─ adata.obsm: 空间坐标
```

**计算空间邻居图**
```
使用 KNN (K-Nearest Neighbors)
  ↓
每个细胞找 4 个最近邻
  ↓
构建邻接矩阵
  ↓
对称化 (确保 A[i,j] = A[j,i])
```

#### 2.3 添加 GP 掩码到数据

```
GP 字典 → 二进制掩码矩阵
  ↓
存储在 adata.obsm 中:
  ├─ nichecompass_gp_targets
  ├─ nichecompass_gp_sources
  ├─ nichecompass_gp_targets_categories
  └─ nichecompass_gp_sources_categories
```

#### 2.4 探索数据

- 查看细胞数量和基因数量
- 可视化细胞类型在空间中的分布

### 第 3 部分：Train Model (训练模型)

#### 3.1 初始化模型

```
NicheCompass 模型架构:
┌─────────────────────────────────────┐
│           输入层                     │
│  基因表达 + 空间邻居图 + GP 掩码    │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│         编码器 (GCN/GAT)            │
│    学习潜在表示 (Latent Space)      │
└─────────────────────────────────────┘
                ↓
┌─────────────────────────────────────┐
│     潜在空间 (~300 维)              │
│   每个维度对应一个 GP 的活性        │
└─────────────────────────────────────┘
                ↓
┌───────────┬───────────┬─────────────┐
│目标基因   │来源基因   │   图解码器  │
│ 解码器    │ 解码器    │             │
└───────────┴───────────┴─────────────┘
```

#### 3.2 训练模型

**损失函数**:
```
Total Loss =
  λ_edge × Edge Loss (重构空间邻居图)
  + λ_gene × Gene Loss (重构基因表达)
  + λ_l1 × L1 Loss (GP 稀疏性)
```

**训练阶段**:
```
阶段 1 (前 25 轮): 预热
  - 使用所有 GP
  - 学习全局模式

阶段 2 (后 375 轮): 主训练
  - 只使用活跃 GP
  - 学习数据特异性模式
```

**训练参数**:
- `n_epochs = 400`: 总训练轮数
- `lr = 0.001`: 学习率
- `lambda_edge_recon = 500000`: 边重构损失权重
- `lambda_gene_expr_recon = 300`: 基因表达重构损失权重
- `edge_batch_size = 1024`: 批次大小

#### 3.3 计算潜在空间的邻居图和 UMAP

```
潜在表示 (高维)
  ↓
计算邻居图 (在潜在空间中)
  ↓
UMAP 降维 (降维到 2D)
  ↓
可视化
```

#### 3.4 保存模型

```
保存内容:
  ├─ 模型参数 (model.pt)
  ├─ 模型配置 (model_kwargs.json)
  ├─ 训练配置 (training_kwargs.json)
  └─ AnnData 对象 (adata.h5ad)

保存位置:
  artifacts/single_sample/{timestamp}/model/
```

### 第 4 部分：Analysis (下游分析)

#### 4.1 可视化潜在 GP 空间

```
创建两个图:
  1. UMAP 图 (潜在空间 2D 投影)
  2. 空间图 (原始物理空间)

目的: 对比潜在空间和物理空间中的细胞类型分布
```

#### 4.2 识别生态位

**Leiden 聚类**:
```
输入: 潜在空间邻居图
  ↓
Leiden 聚类算法
  ↓
输出: 每个细胞的生态位标签

resolution = 0.4:
  - 值越大 → 越多生态位 (更细粒度)
  - 值越小 → 越少生态位 (更粗粒度)
```

#### 4.3 特征化生态位

##### 4.3.1 生态位组成

```
分析每个生态位中不同细胞类型的比例

示例:
生态位 0: 80% 神经元, 20% 胶质细胞
生态位 1: 30% 神经元, 70% 免疫细胞
...
```

##### 4.3.2 差异 GP 分析

**One-vs-Rest 测试**:
```
对于每个生态位:
  该生态位 vs 其他所有生态位
    ↓
测试哪些 GP 显著富集
    ↓
使用 Bayes Factor 评估证据强度
```

**Bayes Factor 解释**:
```
BF > 10:  极强证据 (GP 在该生态位强烈富集)
BF > 5:   强证据
BF > 2.3: 正面证据 (本教程使用的阈值)
BF < -2:  负面证据 (GP 在该生态位活性低)
```

**结果示例**:
```
生态位 | GP 名称              | Bayes Factor
------|---------------------|--------------
  2   | ALCAM_combined_GP  |    11.78
  0   | Add-on_23_GP       |    -9.21

解读:
- ALCAM (细胞黏附) GP 在生态位 2 强烈富集
- 生态位 2 可能是细胞间紧密连接的区域
```

##### 4.3.3 细胞间通信分析

```
选择一个 GP (如 CCK_combined_GP)
  ↓
计算通信网络:
  - 哪些生态位是信号源?
  - 哪些生态位是信号目标?
  - 通信强度如何?
  ↓
可视化网络图
```

**网络图元素**:
- 节点: 生态位
- 边: 细胞间通信
- 边方向: 从源生态位到目标生态位
- 边粗细: 通信强度

## 🎯 快速参考

### 常见问题

**Q1: 训练需要多长时间？**
- 取决于数据规模和硬件
- 示例数据 (9万细胞): 约 1-3 小时 (GPU)

**Q2: 如何调整聚类数量？**
- 修改 `latent_leiden_resolution` 参数
- 值越大 → 越多聚类

**Q3: 如何解释 GP 活性？**
- GP 活性 > 阈值 → 该 GP 在该细胞/生态位中活跃
- 可以查看 GP 的来源基因和目标基因了解其功能

**Q4: 如何选择要分析的 GP？**
- 查看差异 GP 测试结果
- 选择 Bayes Factor 最高的 GP
- 或选择生物学上感兴趣的 GP
